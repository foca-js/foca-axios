# 特性

- 共享相同请求，减少服务器压力
- 请求失败重试，无惧 token 过期
- 缓存响应成功的数据，节省带宽
- 优化 axios 响应结构，加速开发节奏

# 安装

```bash
yarn add axios foca-axios-enhancer
```
