# 特性

- 共享相同的请求，拒绝重复
- 请求失败重试，让服务更稳定
- 缓存成功的数据，减少服务器压力
- 优化 axios 返回值，加速开发节奏
- 专注 Typescript 极致体验

# 安装

```bash
yarn add axios foca-axios-enhancer
```
